Importante lembrar que nos métodos de inserção os campos setados como requeridos não podem ser vazios.

Já nas atualizações não é necessário passar os campos requeridos, já que serão mantidos os existentes caso não sejam passados no corpo da requisição.

Get Methods:

  • deleteOne: filtro para encontrar qual dado irá ser excluído deve ser passado como ReqParam

    Ex:
    Tabela com uma coluna chamada name

    Caso tenha mais colunas e queira utilizar as colunas como parâmetros de busca é só adicionar nos reqParams

    Esse endpoint só permite a deleção de uma linha, caso sejam encontradas mais entradas, retornará uma mensagem avisando.

  • deleteMultipleRows: filtro para encontrar qual dado irá ser excluído deve ser passado como ReqParam

    Mesmo funcionamento do deleteOne, com a diferença que pode realizar a deleção de múltiplas linhas.

  • getAll: Retornará todas as linhas da tabela.

  • getXPTO(NomeDaPagina): passe os filtros por reqParam para buscar os dados desejados.
